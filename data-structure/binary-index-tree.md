# Binary Index Tree

aka Fenwick Tree.

## Motivation

Fenwick tree was proposed to solve the [mutable range sum query problem](https://leetcode.com/problems/range-sum-query-mutable/).

Query: `O(logN)`

Update: `O(logN)`
